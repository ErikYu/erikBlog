  1. 函数也是一种数据；可以将函数值赋给一个变量；
    有两个重要特性：他们包含的是代码；他们是可执行的。要调用某个函数，只需要在名字后面加括号即可；
    命名规则与一般变量相同；
    函数始终有返回值，即便不是显式返回，也会隐式返回一个undefined
  2. 匿名函数：应用于回调函数、即时函数、内部函数

  1. 即时函数：将匿名函数的定义放在一对括号中，最后在接一对括号（传递参数和立即调用）；
    使用即时匿名函数的好处是不会产生任何全局变量；缺点在于无法重复执行；
  2. 内部函数（私有函数）：在函数中定义另外一个函数；
    使用内部函数的好处：确保全局命名空间的纯净；确保私有性；
function a(){
    alert('a');
    return(function(){
        alert('b')
    })
}
var newFunc=a();   //alert('a')
newFunc();         //alert('a')  alert('b')

    描述：如果一个函数在其父级作用域返回之后留住对父级作用域的链接，相关的闭包就会被创建起来。
//闭包1#
var a='glabal varibal';
var F=function(){
    var b='local varible';
    var N=function(){
        var c='inner varible';
        return b;
    }
    return N;
}
b;                 //referenceError, b is not defined
var inner=F();
inner();           //'local variable'

//闭包2#
var inner;
var F=function(){
    var b='local variable';
    var N=function(){
        return b;
    }
    inner=N;
}
F();          //执行F使得N()赋值给了变量inner，该函数升级为全局函数，仍能访问F()作用域
inner();      //执行inner(),输出local variable

//闭包3#
function F(param){
    var N=function(){
        return param;
    }
    param++;
    return N;         //返回函数被调用时，已经执行过一次++了
}
var inner=F(123);
inner();              //124，函数绑定的是作用域本身，而不是函数定义时该作用域的变量和变量返回的值

function F() {

        arr[i]=function () {      //arr[i]拥有的不是值，而是一个引用

}
F();   //返回的arr中三个参数都是function
